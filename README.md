# Object-Motion-Detection
This project implements a simple and effective Object Motion Detection system using Python and OpenCV. The program captures frames from the webcam, processes them, and detects movement by comparing each frame with a dynamically updated background model.

Whenever motion is detected, the moving region is highlighted with bounding boxes, and the status is printed on the screen.
